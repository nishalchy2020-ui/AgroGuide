# AgroGuide

AgroGuide is a smart farming assistant web application built with Flask and SQLite.

## Features
- User registration and login
- Crop recommendation
- Live weather search
- Irrigation advice
- Fertilizer guidance
- Pest and disease help
- User history and saved records

## Technologies Used
- Python
- Flask
- SQLite
- HTML
- CSS
- JavaScript
- Open-Meteo API

## How to Run
1. Create and activate a virtual environment
2. Install requirements:
   pip install -r requirements.txt
3. Run the app:
   python app.py
4. Open:
   http://127.0.0.1:5000

## Project Purpose
This project helps farmers make better decisions using simple digital tools in one platform.
